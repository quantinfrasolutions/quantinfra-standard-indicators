# QuantInfra Standard Indicators

A collection of ready-to-use technical indicators for the QuantInfra platform.

The package is built on top of the QuantInfra SDK and can be used directly in custom strategies or as a reference implementation for creating your own indicators.

## Installation

```bash
dotnet add package QuantInfra.Sdk.StandardIndicators
```

Or via the NuGet Package Manager:

```powershell
Install-Package QuantInfra.Sdk.StandardIndicators
```
